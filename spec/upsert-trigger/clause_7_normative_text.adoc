== Upsert Compatible Triggers Extension Requirement Clauses

Clause 7 defines the requirements for indicating the presence of upsert-compatible RTree index triggers in a GeoPackage.
These requirements are in addition or complement the requirements for RTree indexes as defined by Annex F.3 of the core GeoPackage Encoding Standard Version 1.3.1 or later.

The following requirements class incorporates all the requirements that define the core of the UCTE. These 13 requirements are defined in the sub-clauses to the TIFF Encoding requirements clause.
