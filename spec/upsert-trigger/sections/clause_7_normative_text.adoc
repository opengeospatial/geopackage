== Upsert Compatible Triggers Extension Requirement Clauses

Clause 7 defines the requirements for indicating the presence of upsert-compatible RTree index triggers in a GeoPackage.
These requirements complement the requirements for RTree indexes as defined by Annex F.3 of the core GeoPackage Encoding Standard Version version 1.3.1 and earlier.
Version 1.4.0 of the encoding standard includes these requirements.
