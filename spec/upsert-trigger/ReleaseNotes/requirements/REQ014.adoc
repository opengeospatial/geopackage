|*Requirement 14* {set:cellbgcolor:#CACCCE}| Req/gpkg- pyramid-user-data-datatype-float +
 +

(extends http://www.geopackage.org/spec/#r36[_GPKG-36_] and http://www.geopackage.org/spec/#r37[_GPKG-37_]) For data where the `datatype` column of the corresponding row in the `gpkg_2d_gridded_coverage_ancillary` table is _float_, the `tile_data` BLOB in the http://www.geopackage.org/spec/#tiles_user_tables[_tile pyramid user data table_] containing tiled, gridded data SHALL be of http://www.ietf.org/rfc/rfc2046.txt[_MIME type_] `image/tiff` and the default data encoding SHALL be 32-bit floating point as described in the TIFF Encoding below. {set:cellbgcolor:#FFFFFF}
